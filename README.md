# 公司日常考勤管理系统 java117

## 项目概述

公司日常考勤管理系统是一款基于Springboot+Vue的前后端分离的管理系统。采用MVC三层体系结构，结合Spring、SpringBoot、MyBatis等技术，实现了一套操作便捷、界面简洁的考勤管理系统。

## 技术栈

- 前端：Vue
- 后端：Springboot、Javaee
- 数据库：MySQL
- 构建工具：Maven

## 功能模块

- 用户管理
- 考勤记录
- 请假管理
- 报告生成

## 系统角色

- 系统管理员
- 普通员工

## 运行环境

- Java 1.8+
- MySQL 5.7+
- Maven 3.5+

## 部署步骤

1. 准备所需软件（Java、MySQL、Maven）
2. 导入数据库文件至MySQL
3. 使用Maven编译项目
4. 部署前端和后端代码至服务器

## 目录结构

```
.
├── frontend  # 前端代码
├── backend   # 后端代码
└── database  # 数据库文件
```

## 核心亮点

- 前后端分离，便于管理和扩展
- MVC三层架构，清晰分工
- 结合Springboot、MyBatis，简化开发流程，提高开发效率
